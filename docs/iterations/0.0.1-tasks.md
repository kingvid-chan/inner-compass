# Iteration 0.0.1 — 任务拆解

## 任务列表

### T1: 项目骨架搭建
- **描述**：创建 `backend/main.py`、`backend/requirements.txt`、`static/` 目录结构
- **产出**：目录结构就绪，FastAPI app 可启动
- **验收**：`python backend/main.py` 启动无报错，`/projects/inner-compass/healthz` 返回 200

### T2: FastAPI 静态文件服务与缓存头
- **描述**：实现 FastAPI 挂载 `StaticFiles`，为 HTML 响应添加 `Cache-Control: no-cache` 头
- **产出**：`backend/main.py` 完整
- **验收**：浏览器访问 `/projects/inner-compass/` 返回 index.html，响应头包含 `Cache-Control: no-cache`

### T3: HTML 页面结构
- **描述**：创建 `static/index.html`，包含 Three.js 画布容器、滑块面板、坐标显示面板
- **产出**：`static/index.html`
- **验收**：页面加载后在浏览器中显示画布区域和 7 个滑块

### T4: CSS 样式
- **描述**：创建 `static/css/style.css`，实现左侧滑块面板 + 右侧 3D 画布的布局
- **产出**：`static/css/style.css`
- **验收**：布局合理，滑块和画布同时可见，无遮挡

### T5: Three.js 场景初始化
- **描述**：创建 `static/js/main.js`，初始化 Three.js 场景、相机、渲染器、OrbitControls
- **产出**：`static/js/main.js`（基础框架）
- **验收**：页面显示深色背景的 3D 空白场景，可旋转缩放

### T6: 三维坐标系渲染
- **描述**：在主场景中添加 X/Y/Z 三轴（红/绿/蓝）、刻度标记、轴标签、半透明参考网格面
- **产出**：`static/js/main.js`（坐标系统）
- **验收**：场景中可见三轴、刻度、标签、网格面

### T7: 参数滑块绑定
- **描述**：创建 `static/js/sliders.js`，实现 7 个滑块的 DOM 绑定和公式计算（S×L×Iteration, C², W−F×Delta）
- **产出**：`static/js/sliders.js`
- **验收**：滑块拖动时控制台输出正确的计算坐标

### T8: 球体位置实时更新
- **描述**：连接 sliders.js 和 main.js，滑块变化时实时更新 3D 场景中球体位置
- **产出**：连线逻辑
- **验收**：拖动任意滑块，球体在三维坐标系中实时移动

### T9: 坐标显示面板
- **描述**：在页面上显示当前 X/Y/Z 坐标数值（含中文标签）
- **产出**：更新坐标显示逻辑
- **验收**：拖动滑块时坐标面板数值实时更新

### T10: integrity check
- **描述**：自测完整流程 — 启动后端、浏览器访问、验证所有功能
- **产出**：`evidence/claude/self_test-0.0.1.json`、`evidence/claude/handoff-0.0.1.json`
- **验收**：`ready_for_verification: true`
