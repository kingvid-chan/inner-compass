# Iteration 0.0.1 — 自我内在评价三维坐标系

## 需求概述

构建一个交互式 Web 3D 工具，用户通过调节 7 个内在参数，实时看到自己在三维坐标系中的位置，用于自我反思与评价。

## 三轴定义

| 轴 | 标签 | 公式 | 范围 |
|---|---|---|---|
| X | 认知实践轴 | (S × L) × Iteration | 1–1000 |
| Y | 连接轴 | C² | 1–100 |
| Z | 心灵自由轴 | W − F × Delta | −90–10 |

## 参数

| ID | 标签 | 默认值 | 范围 |
|---|---|---|---|
| S | 感知力 | 5 | 1–10 |
| L | 逻辑力 | 5 | 1–10 |
| Iteration | 迭代系数 | 5 | 1–10 |
| C | 连接深度 | 5 | 1–10 |
| W | 愿力 | 5 | 1–10 |
| F | 恐惧 | 5 | 1–10 |
| Delta | 分别心系数 | 5 | 1–10 |

## 核心功能

1. Three.js 三维坐标系（WebGL，可旋转/缩放）
2. 7 个滑块实时联动坐标点
3. 轴标签/刻度/网格参考面
4. 当前坐标数值面板
5. 可选：历史记录（localStorage）、截图导出

## 技术栈

- 前端：HTML/CSS/JS + Three.js CDN
- 后端：FastAPI 静态文件服务 + healthz
- 部署：aliyun-cowork Nginx subpath
